# Netflix-Clone 项目验收文档

## 1. 项目概述

**项目名称**：Netflix-Clone
**项目类型**：iOS应用程序
**项目简介**：这是一个模仿Netflix应用界面和功能的iOS项目，使用Swift和UIKit框架开发，通过TMDB API获取电影和电视节目数据。
